◆ boundaryNodeList

std::vector<int> Mesh::boundaryNodeList

A list of the degree of freedom that are at boundary.

Note
Actually this is not NODE list, but degree of freedom list, e.g., if node 1,2 are X&Y fixed, node 4 is x fixed, then the DOF list is (2,3 | 4,5 | 8) and the boundary value list below is (0,0,0,0,0).
